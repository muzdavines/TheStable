Sword & Shield Animset Pro - Installation
-----------------------------------------

1. Create a new, empty Unity Project
2. Import PlayMaker
3. Import Sword & Shield Animset Pro
4. Unzip the Assets\SwordShieldAnimsetPro\InputManager_Replace\InputManager.zip
5. Replace your existing InputManager.asset (it's in ProjectSettings\ folder) with the new, unzipped InputManager.asset file

Done :)
